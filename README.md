# Web Application Base v0.1.0


# How to install?

```
make install
make migrate
make loaddata
```

> loaddata: Create an admin user with email 'victor@yopmail.com' and password 'admin'.


# How to start?

```
make start
```

# How to stop?

```
make stop
```
